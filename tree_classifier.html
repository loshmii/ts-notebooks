Classes

Just like in KNN classificator, we use Spec and Config classes in order to specify hyperparamater grid over which we test different classificators. We define DecisionTree grid over the following hyperparameters:

  • Tree depth
  • Minimal number of samples per leaf
  • Criterion entropy and gini

Cross-validation process is identical to the one in KNN classificator.

    num_cols=['Age','ALB','ALP','ALT','AST','BIL','CHE','CHOL','CREA','GGT','PROT'],
